## ⚙️ Configuration

Settings are stored in `config.json` at `%APPDATA%\PixelGun3DTrainer\config.json`.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `menu_hotkey` | String | `"Insert"` | Key to toggle menu |
| `auto_attach` | Boolean | `true` | Auto-attach on startup |
| `process_name` | String | `"PixelGun3D.exe"` | Target process |
| `esp_enabled` | Boolean | `true` | Enable ESP overlay |
| `aimbot_enabled` | Boolean | `false` | Enable aimbot |
| `safe_mode` | Boolean | `true` | Restricts unstable features |
| `log_level` | String | `"info"` | `"debug"`, `"info"`, `"error"` |
